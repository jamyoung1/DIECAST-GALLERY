DIECAST-GALLERY

## 요구사항

   * Main - 상품 목록, 가격, 할인가 등 상품의 목록 정보.

   * Cart - 장바구니
   
   * Login - email, password로 로그인 가능
   
   * Order - 주문 페이지. 유저가 담은 상품의 목록, 총 가격, 결제 수단 선택 
   
   * Orders - 유저가 주문한 상품 목록 조회


## 1. 사용 기술
  * Spring-Boot (SDK 11)
  * JPA
  * Mysql
  * BootStrap(v5.1.3)
  * lombok
  * spring web
  
## 2. 주요 기능
  * 상품 목록 
  * 상품 장바구니 담기
  * 상품 결제
  * 상품 결제 후 회원의 결제 목록 조회
  * 로그인, 로그아웃
